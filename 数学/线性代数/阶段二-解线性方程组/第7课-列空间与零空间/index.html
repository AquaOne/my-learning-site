    第7课：列空间与零空间¶

    一句话直觉：列空间回答"我们能到哪儿？"，零空间回答"哪些方向被压缩成了原点？"——这两个问题定义了矩阵的全部行为。

    1. 向量空间¶

    进入列空间和零空间之前，先理解什么是"空间"。

    定义：一个向量空间是一个集合，里面包含向量，并且对加法和数乘封闭（即：空间中的任意两个向量相加，结果还在空间中；一个向量乘以任意标量，结果还在空间中）。

    最简单的例子：\(\mathbb{R}^2\)¶

    • 所有二维向量 \((x, y)\) 组成的集合
    • 两个这样的向量相加，结果仍然是二维向量 ✓
    • 乘以标量，结果仍然是二维向量 ✓

    子空间¶

    向量空间的子集，本身也是一个向量空间（即对加法和数乘封闭）。

    \(\mathbb{R}^2\) 的子空间： 1. 整个 \(\mathbb{R}^2\)（最大的子空间） 2. 过原点的直线（最小的非平凡子空间） 3. 原点 \(\{\vec{0}\}\)（最小的子空间）

    注意：不过原点的直线不是子空间——因为数乘会把向量移出这条线（原点不在线上）。

    矩阵定义的两个关键子空间¶

    对于 \(m \times n\) 矩阵 \(A\)： - 列空间 \(C(A)\)：\(\mathbb{R}^m\) 的子空间 - 零空间 \(N(A)\)：\(\mathbb{R}^n\) 的子空间

    2. 列空间 Column Space \(C(A)\)¶

    定义¶

    \(A\) 的列空间是所有列向量的线性组合构成的集合： $\(C(A) = \{\text{所有 } A\vec{x} \mid \vec{x} \in \mathbb{R}^n\}\)$ 或者说：\(C(A) = \text{span}(\text{列}_1, \text{列}_2, \ldots, \text{列}_n)\)

    为什么重要？¶

    回到 \(A\vec{x} = \vec{b}\)：

    \(A\vec{x} = \vec{b}\) 有解的充要条件是 \(\vec{b} \in C(A)\)。

    例子¶

    \[A = \begin{bmatrix}1 & 2 \\ 3 & 6\end{bmatrix}\]

    列向量：\(\begin{bmatrix}1 \\ 3\end{bmatrix}\) 和 \(\begin{bmatrix}2 \\ 6\end{bmatrix} = 2\begin{bmatrix}1 \\ 3\end{bmatrix}\)。二者共线。

    \[C(A) = \{c\begin{bmatrix}1 \\ 3\end{bmatrix} \mid c \in \mathbb{R}\}\]

    这是一条经过原点的直线（在 \(\mathbb{R}^2\) 中）。

    所以 \(\begin{bmatrix}1 \\ 4\end{bmatrix}\) 是否在 \(C(A)\) 中？不在——它不在 \(\begin{bmatrix}1 \\ 3\end{bmatrix}\) 的方向上。

    3. 零空间 Nullspace \(N(A)\)¶

    定义¶

    \(A\) 的零空间是所有满足 \(A\vec{x} = 0\) 的 \(\vec{x}\) 构成的集合： $\(N(A) = \{\vec{x} \in \mathbb{R}^n \mid A\vec{x} = \vec{0}\}\)$

    为什么叫零空间？¶

    因为矩阵 \(A\) 把这些向量"压缩"成了零。零空间中的向量就是被 \(A\)消灭的方向。

    验证是子空间¶

    如果 \(A\vec{x} = 0\) 且 \(A\vec{y} = 0\)，那么： - \(A(\vec{x} + \vec{y}) = A\vec{x} + A\vec{y} = 0 + 0 = 0\) ✓ - \(A(c\vec{x}) = cA\vec{x} = c \cdot 0 = 0\) ✓

    所以零空间对加法和数乘封闭。

    例子¶

    \[A = \begin{bmatrix}1 & 2 \\ 3 & 6\end{bmatrix}\]

    求 \(N(A)\)：解 \(A\vec{x} = 0\)

    \[\begin{cases}x + 2y = 0 \\ 3x + 6y = 0\end{cases}\]

    第二个方程是第一个的 3 倍，实际上只有一个独立方程：\(x + 2y = 0\)。

    所以：\(N(A) = \{c\begin{bmatrix}-2 \\ 1\end{bmatrix} \mid c \in \mathbb{R}\}\)

    这是一条直线。几何上说：\(A\) 把沿着 \(\begin{bmatrix}-2 \\ 1\end{bmatrix}\) 方向的所有向量都压缩成了 0。

    4. 列空间 vs 零空间：关键对比¶

    列空间 \(C(A)\) 零空间 \(N(A)\)
    在哪个空间里？ \(\mathbb{R}^m\)（和行数相同） \(\mathbb{R}^n\)（和列数相同）
    定义 \(\{A\vec{x} \mid \vec{x} \in \mathbb{R}^n\}\) \(\{\vec{x} \mid A\vec{x} = 0\}\)
    回答的问题 \(A\vec{x} = \vec{b}\) 什么时候有解？ 哪些向量被 \(A\) 变成零？
    直观 矩阵列向量张成的空间 解 \(A\vec{x} = 0\) 得到的所有解

    5. 例题实战¶

    例题 1¶

    \[A = \begin{bmatrix}1 & 0 & 0 \\ 0 & 1 & 0\end{bmatrix}\]

    列空间：\(C(A) = \text{span}(\begin{bmatrix}1 \\ 0\end{bmatrix}, \begin{bmatrix}0 \\ 1\end{bmatrix}) = \mathbb{R}^2\)（整个平面）

    零空间：解 \(A\vec{x} = 0\)： $\(\begin{bmatrix}1 & 0 & 0 \\ 0 & 1 & 0\end{bmatrix}\begin{bmatrix}x \\ y \\ z\end{bmatrix} = \begin{bmatrix}0 \\ 0\end{bmatrix}\)$

    \(x = 0, y = 0, z\) 任意。\(N(A) = \{\begin{bmatrix}0 \\ 0 \\ c\end{bmatrix} \mid c \in \mathbb{R}\}\) — 就是 \(z\) 轴。

    几何直觉：这个矩阵把 \(\mathbb{R}^3\) 投影到了 \(xy\) 平面（\(z\) 轴方向的信息丢失了），所以 \(z\) 轴方向的所有向量都变成了 0。

    例题 2¶

    \[A = \begin{bmatrix}1 & 1 \\ 2 & 2 \\ 3 & 3\end{bmatrix}\]

    列空间：两个列向量共线，\(C(A) = \{c\begin{bmatrix}1 \\ 2 \\ 3\end{bmatrix} \mid c \in \mathbb{R}\}\)（\(\mathbb{R}^3\) 中的一条线）

    零空间：解 \(A\vec{x} = 0\)： $\(\begin{cases}x + y = 0 \\ 2x + 2y = 0 \\ 3x + 3y = 0\end{cases}\)$

    \(x + y = 0\) → \(y = -x\)。\(N(A) = \{c\begin{bmatrix}1 \\ -1\end{bmatrix} \mid c \in \mathbb{R}\}\)（\(\mathbb{R}^2\) 中的一条线）
